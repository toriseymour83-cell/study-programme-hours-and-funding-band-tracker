# Original approach

This folder shows the original version of a study programme planning tool I created to support post-16 funding band decisions.

The aim was to help staff plan a student's weekly timetable, calculate their total planned hours across the academic year, and use that information to identify the correct funding band.

The original approach was built around a spreadsheet-based timetable and a learning agreement document. Staff could use the timetable to select or enter planned activities for each day and time slot, and the spreadsheet would calculate the total number of planned hours.

This total could then be used alongside the learning agreement as evidence of the planned study programme.

This approach worked well as a practical operational tool because it was easy for staff to use and aligned with the documents already used in the organisation. It supported planning, consistency, and clearer recording of learning hours.

However, the process relied on manual input and document handling, which meant it was less scalable, harder to validate, and more difficult to analyse across multiple students or employers.

The files in this folder represent that original spreadsheet and document-based solution.
