
# Modern approach

This folder shows how I would redesign the original study programme hours and funding band process using a more structured data workflow.

Instead of relying on an embedded spreadsheet inside individual learning agreements, the modern approach would separate the process into clear input, calculation, and reporting stages.

A timetable or session plan would still be the starting point, but the data would be stored in a structured format so that planned hours could be calculated consistently across all learners. Funding bands could then be assigned automatically using rules applied to the calculated annual hours.

This would make the process easier to validate, easier to report on, and easier to scale across multiple students, employers, or cohorts.

The aim of the modern version is to show how the same operational problem could now be handled using cleaner data structures, automated transformations, and management reporting.
